# 🚀 Naukri Auto Applier

Automated job application bot using Playwright that applies to relevant jobs on Naukri based on predefined filters and answers.

---

## 📌 Overview

This project automates the job application process by:

- Searching jobs using predefined keywords
- Applying filters like experience, location, etc.
- Automatically answering screening questions
- Avoiding duplicate applications using a database

---

## 🧠 Key Features

- 🔍 Keyword-based job search
- 🤖 Automated job application using Playwright
- 🧾 Smart screening question handling
- 🗂️ SQLite database to track applied jobs
- 🔁 Scheduler support for periodic runs
- ❌ Duplicate job prevention
